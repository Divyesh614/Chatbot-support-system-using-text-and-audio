# Chatbot Support System

A simple AI chatbot support system using:

- HTML
- CSS
- JavaScript
- Text Chat
- Voice Recognition
- Speech Response

## Features

✅ Text-based chatbot  
✅ Audio voice input  
✅ Voice response  
✅ Simple user interface  

## Technologies Used

- HTML
- CSS
- JavaScript
- Web Speech API

## How to Run

1. Download the project
2. Open folder in VS Code
3. Run `index.html`
